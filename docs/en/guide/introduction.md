# Introduction

> **BokeBox** — content in, AI podcasts out  
> Repo: <https://github.com/vastsa/BokeBox>  
> License: LGPL-3.0-only

## In one line

Drop videos, links, articles, meeting notes, course materials, or plain drafts into the box. BokeBox will:

1. Understand the content  
2. Write a spoken script  
3. Speak it in the voice you choose  
4. Turn it into a private podcast you can play anytime  

Not another single-format “media-to-audio” converter — a **multi-source personal AI podcast studio**.

## 30-second pitch

```text
  What you put in                    What BokeBox gives back
 ─────────────────                  ──────────────────────
  Meetings / notes                   Rhythmic spoken episodes
  Course replays / materials ─AI──▶  Custom host personas
  Long-form articles / drafts        Preset / described voices
  Any link (plugin-extensible)       Cover · flashcards · progress
```

## Why it matters

| You care about | How BokeBox does it |
| --- | --- |
| **Finish listening** | Spoken structure: open, highlights, close |
| **Any input** | Video / link / draft / meetings & courses; plugins extend |
| **Sounds human** | Natural voices + delivery tags |
| **You own persona** | Host, audience, style, show name — global or per episode |
| **AI-native** | MCP for create / query from Cursor, Claude, … |
| **Knowledge sticks** | Auto flashcards |
| **Data stays local** | Single-user private deploy |

## Who it’s for

- People whose “read later” never happens  
- Anyone who digests long content while commuting or doing chores  
- Makers who want a show “just for me” without a mic  
- Solo users / small teams who need self-hosting
